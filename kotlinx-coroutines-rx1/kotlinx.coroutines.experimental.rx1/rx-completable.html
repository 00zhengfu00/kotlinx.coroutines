rxCompletable

fun rxCompletable(
    context: CoroutineContext,
    block: suspend CoroutineScope.() -> Unit
): Completable (source)

Creates cold Completable that runs a given block in a coroutine. Every time the returned completable is subscribed, it starts a new coroutine in the specified context. Unsubscribing cancels running coroutine.

Coroutine action Signal to subscriber
Completes successfully onCompleted
Failure with exception or unsubscribe onError
