delay

open suspend fun delay(
    time: Long,
    unit: TimeUnit = TimeUnit.MILLISECONDS
): Unit (source)

Delays coroutine for a given time without blocking a thread and resumes it after a specified time. This suspending function is cancellable. If the Job of the current coroutine is completed while this suspending function is suspended, this function immediately resumes with CancellationException.
