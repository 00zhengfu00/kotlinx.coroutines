onAwait

abstract fun <T> Deferred<T>.onAwait(
    block: suspend (T) -> R
): Unit (source)

Clause for Deferred.await suspending function that selects the given block with the deferred value is resolved. The select invocation fails if the deferred value completes exceptionally (either fails or it cancelled).
