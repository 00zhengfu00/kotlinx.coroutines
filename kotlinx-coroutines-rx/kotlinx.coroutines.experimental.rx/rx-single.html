Run asynchronous computations based on block coroutine parameter

Execution starts immediately within the ‘asyncRx’ call and it runs until the first suspension point is reached (‘*await’ call for some Observable instance). Remaining part of coroutine will be executed as it’s passed into ‘subscribe’ call of awaited Observable.

Parameters

block - a coroutine representing reactive computations

Return Observable with single value containing expression returned from coroutine
