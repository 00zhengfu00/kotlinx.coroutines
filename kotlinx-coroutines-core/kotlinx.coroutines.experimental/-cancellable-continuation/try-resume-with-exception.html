tryResumeWithException

abstract fun tryResumeWithException(
    exception: Throwable
): Any? (source)

Tries to resume this continuation with a given exception and returns non-null object token if it was successful, or null otherwise (it was already resumed or cancelled). When non-null object was returned, completeResume must be invoked with it.
