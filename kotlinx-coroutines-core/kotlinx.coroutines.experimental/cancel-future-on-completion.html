cancelFutureOnCompletion

fun Job.cancelFutureOnCompletion(
    future: Future<*>
): Registration (source)

Cancels a specified future when this job is complete. This is a shortcut for the following code with slightly more efficient implementation (one fewer object created).

onCompletion { future.cancel(true) }
