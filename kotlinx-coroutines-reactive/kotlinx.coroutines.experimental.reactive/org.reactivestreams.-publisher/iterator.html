iterator

operator fun <T> Publisher<T>.iterator(): ChannelIterator<T> (source)

Subscribes to this Publisher and returns an iterator to receive elements emitted by it.

This is a shortcut for open().iterator(). See open if you need an ability to manually unsubscribe from the observable.
