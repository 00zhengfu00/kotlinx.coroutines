rxSingle

fun <T> rxSingle(
    context: CoroutineContext,
    block: suspend CoroutineScope.() -> T
): Single<T> (source)

Creates cold Single that runs a given block in a coroutine. Every time the returned single is subscribed, it starts a new coroutine in the specified context. Coroutine returns a single value. Unsubscribing cancels running coroutine.

Coroutine action Signal to subscriber
Returns a value onSuccess
Failure with exception or unsubscribe onError
