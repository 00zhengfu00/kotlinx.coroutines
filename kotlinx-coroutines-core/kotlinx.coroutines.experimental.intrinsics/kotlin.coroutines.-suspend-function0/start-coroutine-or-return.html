startCoroutineOrReturn

fun <T> SuspendFunction0<T>.startCoroutineOrReturn(
    completion: Continuation<T>
): Any? (source)

Starts coroutine without receiver and with result type T. This function creates and start a new, fresh instance of suspendable computation every time it is invoked. If the coroutine never suspends, then its result is returned directly, otherwise it returns COROUTINE_SUSPENDED and the completion continuation is invoked when coroutine completes.
