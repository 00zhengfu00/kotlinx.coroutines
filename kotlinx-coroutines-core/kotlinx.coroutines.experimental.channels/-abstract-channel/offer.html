offer

fun offer(element: E): Boolean (source)

Adds element into this queue if it is possible to do so immediately without violating capacity restrictions and returns true. Otherwise, it returns false immediately or throws ClosedSendChannelException if the channel isClosedForSend normally. It throws the original close cause exception if the channel has failed.
